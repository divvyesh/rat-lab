## 🧬 **ML/STATISTICAL METHODS EXPLAINED:**

### **1. K-Means Clustering**
```python
# Pseudocode
for each participant:
    calculate sentiment_score = positive_count / total_questions
    
classify into 3 clusters:
    Enthusiasts: sentiment_score > 0.66
    Cautious: 0.33 < sentiment_score ≤ 0.66
    Skeptics: sentiment_score ≤ 0.33
```

**Business Value:** Identifies behavioral archetypes for targeted messaging

---

### **2. Descriptive Statistics**
```python
# Sample Mean
μ = Σ(all_numeric_responses) / n

# Standard Deviation
σ = sqrt(Σ(xi - μ)² / n)

# Variance
σ² = Σ(xi - μ)² / n
```

**Business Value:** Validates data quality and response reliability

---

### **3. Sentiment Analysis**
```python
for each response:
    analyze text sentiment
    classify: Positive | Neutral | Negative
    
aggregate:
    positive_pct = positive_count / total
    neutral_pct = neutral_count / total
    negative_pct = negative_count / total
```

**Business Value:** Emotional tone of responses informs messaging strategy

---

### **4. Correlation Analysis**
```python
for each segment:
    calculate avg_positive_sentiment
    
compare across segments:
    identify which segments respond most favorably
    
correlation_matrix:
    segment × outcome correlation
```

**Business Value:** Identifies which segments resonate with concept

---

### **5. Regression Analysis** (Analysis Tab)
```python
X = behavioral_traits (risk_aversion, price_sensitivity, etc.)
y = outcome_scores (intent, satisfaction, etc.)

model = LinearRegression()
model.fit(X, y)

for each participant:
    plot(trait_value, outcome_score)
    color_by_segment
    
identify:
    - Correlation coefficient (r)
    - Statistical significance (p-value)
    - Outliers
```

**Business Value:** Predicts which traits drive positive outcomes

---

### **6. Hypothesis Testing** (Analysis Tab)
```python
hypothesis: "Integration mentions predict higher intent"

group_A = participants who mentioned "integration"
group_B = participants who didn't

t_test(group_A.scores, group_B.scores)
→ p_value, effect_size, statistical_significance

if p_value < 0.05:
    hypothesis = VALIDATED ✓
```

**Business Value:** Statistical proof for business decisions
